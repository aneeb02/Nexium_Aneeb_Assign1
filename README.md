#  Quote Generator

A simple, elegant web app that lets users pick a topic and get an inspiring quote â€” with support for dark mode, transitions, and custom styling.

Built using:
- Next.js 15 (App Router)
- TypeScript
- TailwindCSS
- shadcn/ui
- Framer Motion (for animations)
- next-themes (for theme switching)
---

## Features

-  10+ curated topics with multiple quotes each
-  Light / Dark mode support (with smooth transition)
-  Styled using both shadcn/ui and daisyUI
-  Form validation with React Hook Form + Zod
-  Smooth animations using Framer Motion
